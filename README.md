### DEV SETUP

##### Step 1. Download codebase.

```
git clone https://github.com/aman-webdeveloper/Job-Recommender.git

cd Job-Recommender
```

##### Step 2. Install Dependencies.

```
    npm i
```

##### Step 3. Run Backend.

```
    npm run server
```

##### Step 4. Run Frontend.

> In dedicated terminal.

```
    npm run dev
```
